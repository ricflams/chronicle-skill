# Visuals Directory

This directory contains diagrams, sketches, mood boards, and other visual artifacts for the project.

## Contents

- Architecture diagrams (Mermaid .mmd files or images)
- Workflow diagrams
- Sketches and mockups
- Screenshots of key moments
- Photos of physical artifacts (for DIY projects)

## Formats

- **Diagrams:** .mmd (Mermaid), .svg, .png
- **Photos:** .jpg, .png
- **Sketches:** Any image format

All visuals should be referenced from other chronicle files for context.
